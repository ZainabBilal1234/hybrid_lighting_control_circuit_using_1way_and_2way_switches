# 📌 Project Overview

This project demonstrates a hybrid lighting control circuit using both **1-way** and **2-way switches** to operate two lamps through a combination of **series and parallel wiring**.

The design reflects real-world residential and commercial electrical systems commonly used in hallways, staircases, and room lighting applications.

---

# 📖 Description

The circuit was designed to allow independent control of two lamps:

* Lamp 1 is controlled using a **1-way switch**
* Lamp 2 is controlled using a **2-way traveler switching system**

The hybrid configuration improves flexibility, safety, and practical usability while demonstrating important electrical wiring concepts.

The project was physically assembled and tested successfully using standard electrical components.
